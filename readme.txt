== Description ==

Based on the codecogs equation editor for tinymce 4.x. It returns
the functionality that was present in ultimate tinymce plugin
for the up-to-date tinymce and tinymce advanced editors of
Wordpress 4.x versions.

== Installation ==

The installation of WP Mathedit is easy.

1. Upload the wp-mathedit.zip to your Plugins directory and unzip
2. Activate the plugin through the 'Plugins' menu in WordPress

== How to use ==

Go to a new post, page or custom post type. The tinymce editor will have a new icon (Fx) which will open the codecogs editor.
For further info visit Codecogs page: http://www.codecogs.com/latex/integration/tinymce_v4/install.php
